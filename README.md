# A Parametric and Fully 3D-Printable Wing

## Basic design parameters
### - Chord Length ($c$)
### - Aspect Ratio ($AR$)
### - Wingspan ($b=AR*c$)

## Advanced aerodynamic parameters
### - Taper ratio ($0-1$)
### - Sweep angle $\phi$ (measured from a horizontal axis)
### - Dihedral angle $\beta$

## Mechanical parameters
### - Rib angle $\theta$
#### - Rib offset $ D_{rib} = \frac{2c \cos\left(\theta\right)}{3 \sin\left(\theta\right)} $
#### - Rib count
